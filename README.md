# 🎨 AI Thumbnail Creator (MERN Stack)

Create stunning, high-converting thumbnails using AI. This project is a full-stack MERN application that allows users to generate different styles of thumbnails with authentication and payment integration.

---

## 🚀 Features

* 🤖 AI-powered thumbnail generation
* 🎭 Multiple thumbnail styles (YouTube, Gaming, Cinematic, Clickbait, etc.)
* 🔐 Google Authentication (OAuth 2.0)
* 💳 Payment Integration (Dodo Payments)
* 🧠 Prompt-based thumbnail customization
* 📱 Fully responsive UI
* ⚡ Real-world MERN stack architecture

---

## 🛠️ Tech Stack

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/ai-thumbnail-creator.git
cd ai-thumbnail-creator
```

### 2. Install dependencies

```bash
# install frontend
cd client
npm install

# install backend
cd ../server
npm install
```

### 3. Environment Variables

Create a `.env` file in the server folder and add:

```
MONGO_URI=your_mongodb_connection
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_secret
AI_API_KEY=your_ai_api_key
DODO_PAYMENT_KEY=your_payment_key
```

---

### 4. Run the app

```bash
# start backend
npm run server

# start frontend
npm start
```
